# mastersproject
Here i keep my scripts related to my master's project at UNIGE.
The main features are:
- Script_1000genomes.sh: A shell script to dowload data from the 1000 genomes project(1KGP) from the 3 release. This script allows selection of loci and continent.
- Script_rsID.sh: A shell script to annotate VCF riles with rsID (secondary priority so still not fully implemented).

Soon another branch will be added to access to	data based on the high-coverage WGS Illumina data from 3,202 1000 Genomes Project (1kGP) samples, described in Byrska-Bishop et al., 2022 (PMID: 36055201). Especicially the one in the https://ftp.1000genomes.ebi.ac.uk/vol1/ftp/data_collections/1000G_2504_high_coverage/working/20190425_NYGC_GATK directory of their ftp site.
